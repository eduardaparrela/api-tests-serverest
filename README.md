
#API Tests Postman 

Projeto de testes de API utilizando Postman e a API ServeRest, foco em validação de endpoints e autenticação, simulando cenários reais de um sistema e-commerce.


#Tecnologias utilizadas
Postman
Serverest - API REST
JavaScript para scripts de teste

#Fluxos testados 
-Cadastro de usuario
-Login (autenticacao)
-Criacao de produto
-Listagem de produto
-Exclusao de produto


#Cenarios testados: 
Positivos:
Criar usuario
Login com sucesso
Criar produto com token
Deletar produto com token

Negativos: 
Criar usuário com email duplicado
Criar produto sem token
Deletar produto sem token 
Deletar produto com ID inexistente
